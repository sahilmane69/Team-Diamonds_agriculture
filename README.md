**Krishimitra AI** is an intelligent agriculture assistant that helps farmers take the best action for their crops using AI-driven insights.

It supports **native language text input** and **camera-based crop analysis** to provide simple, practical recommendations.

---

##  What It Does
- Understands farmer queries in local languages
- Analyzes crop images from a camera
- Suggests actionable steps for crop health and yield
- Makes AI accessible for real-world agriculture

---

##  How It Works
1. Farmer enters a query or uploads a crop image
2. AI analyzes the input
3. Best possible action is recommended
